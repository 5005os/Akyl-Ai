# 🇰🇬 AI Kyrgyzstan — национальный ИИ-ассистент

Прототип сайта с искусственным интеллектом о Кыргызстане: умный чат,
история, прогнозы развития и проверенные источники.

![flag](https://img.shields.io/badge/Made%20for-Kyrgyzstan-e30613)

## ✨ Возможности

- 🤖 **ИИ-чат** — отвечает на вопросы о Кыргызстане (история, Иссык-Куль, Манас, экономика).
  - **Офлайн-режим** — встроенная база знаний, работает без интернета и без ключей.
  - **Онлайн-режим** — подключение к модели Claude по API-ключу для развёрнутых ответов.
- 📜 **История** — интерактивная лента событий от древних кыргызов до наших дней.
- 🔮 **Прогнозы** — перспективы по экономике, IT, энергетике, туризму, климату.
- 📚 **Источники** — официальные ссылки (Нацстатком, Всемирный банк, ЮНЕСКО и др.).

## 🚀 Как запустить

Сайт полностью статический — никакой установки не нужно.

**Вариант 1 — просто открыть файл:**
```
Откройте index.html в браузере (двойной клик)
```

**Вариант 2 — локальный сервер (рекомендуется):**
```bash
# Python 3
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## 🔑 Подключить настоящий ИИ (Claude)

1. Получите API-ключ на [console.anthropic.com](https://console.anthropic.com).
2. На сайте откройте раздел чата → «⚙️ Подключить настоящий ИИ».
3. Вставьте ключ и нажмите «Сохранить». Ключ хранится **только в вашем браузере**.

> ⚠️ Для продакшена API-ключ нельзя держать в браузере — нужен серверный
> прокси. Текущий вариант — для прототипа и личного использования.

## 🛠️ Как добавить новые знания

Откройте `data/knowledge.js` и дополните массивы:
- `history` — события истории;
- `forecasts` — прогнозы;
- `sources` — источники;
- `qa` — пары «ключевые слова → ответ» для офлайн-чата.
