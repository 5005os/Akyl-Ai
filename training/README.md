# 🧠 Обучение своей модели AkylAi

Здесь всё, чтобы создать **свою ИИ-модель** Кыргызстана — бесплатно, в Google Colab.

## 📁 Файлы

| Файл | Что делает |
|------|-----------|
| `prepare_dataset.py` | Собирает все данные из `dataset/` в один файл для обучения |
| `train_akylai.ipynb` | Google Colab-ноутбук: обучает модель Qwen на твоих данных |
| `akylai_dataset.jsonl` | Итоговый датасет (создаётся скриптом) |
| `prepare_mlx.py` | Готовит данные для обучения на Mac (MLX) |
| `build_qa_bank.py` | Собирает все Q&A в `data/qa_bank.js` — умный поиск на сайте |
| `MLX_MAC.md` | Инструкция: обучение на Mac (Apple Silicon), без облака |

## 🖥️ Обучение на Mac (без облака)

Есть Mac с чипом M1/M2/M3/M4? Можно обучать **прямо на компьютере**, без Colab
и без лимитов. Пошаговая инструкция — в [`MLX_MAC.md`](MLX_MAC.md).

## 🚀 Как обучить свою модель (пошагово)

### Открыть ноутбук одной кнопкой

[![Открыть в Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/5005os/Cloud-Code/blob/main/training/train_akylai.ipynb)

Или ссылкой: **https://colab.research.google.com/github/5005os/Cloud-Code/blob/main/training/train_akylai.ipynb**

### Дальше:

1. Войди в свой **Google-аккаунт (Gmail)**, если попросит.
2. Включи бесплатный GPU: **Среда выполнения → Сменить среду выполнения → `T4 GPU` → Сохранить**.
3. Нажимай ▶ на каждом блоке **по порядку, сверху вниз** — жди зелёную галочку ✅, потом следующий.
4. Блок 6 (обучение) — самый долгий, ~30–50 минут. Не закрывай вкладку.
5. Блок 9 сам скачает готовую модель `akylai_model.zip` в твои «Загрузки». 🎉

💡 Если какой-то блок покажет **красную ошибку** — скопируй её текст и пришли Claude, он подскажет.

## 💾 Что получишь

- **LoRA-адаптер** (`akylai_lora`) — маленький файл с «кыргызскими знаниями»
- **GGUF-модель** (`akylai_model/*.gguf`) — чтобы запускать локально в **Ollama**

## ▶️ Запуск своей модели локально (Ollama)

```bash
# 1. Установи Ollama: https://ollama.com
# 2. Создай Modelfile рядом с .gguf файлом:
#    FROM ./akylai_model.gguf
# 3. Собери и запусти:
ollama create akylai -f Modelfile
ollama run akylai
```

## 🌱 Как сделать модель умнее

Чем больше и качественнее данные — тем лучше модель:
1. Пополняй файлы в `dataset/` (переводы, статьи, Q&A).
2. Снова запусти `prepare_dataset.py` и обучение.
3. Повторяй — модель будет улучшаться с каждым разом.

## 🎯 Базовая модель

По умолчанию используется **Qwen2.5-3B** — она хорошо знает много языков и
помещается в бесплатный GPU. Можно поменять на Llama 3 или Gemma в ячейке №3.
